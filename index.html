Connect Firebase
One-time setup to enable cloud storage
How to get your Firebase config:
1. Go to console.firebase.google.com → Create a project
2. Add a Web App → copy the firebaseConfig object
3. Enable Authentication → Email/Password
4. Create a Firestore Database (start in test mode)
5. Paste your values below and click Connect
Create Admin Account
This admin account will be the first user. After setup, add riders and managers from the Admin dashboard.
Firestore Security Rules
Copy this into Firebase Console → Firestore → Rules:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }
